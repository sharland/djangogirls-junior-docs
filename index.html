Installation instructions

Django is written using the Python programming language and therefore editing it will require the use of software which can create Python scripts.

The next slides will look at installation instructions for your own machine. If you are working through this tutorial in a school computer lab it is likely that python should have been already installed.

Installation on macOS

Note: Before you install Python on macOS you should ensure that your Mac settings allow for installing packages that aren't from the App store. Go to System Preferences (it's in the Applications folder), click Security and Privacy and then the General tab. If your "Allow apps downloaded from:" is set to "Mac App store" change it to "Mac App Store and identified developers"

Go to Python.org and select the latest version 3 release for Python. This will download a pkg file to your Mac. Once it is downloaded double click this file to start installation. A new Python application will be added to your applications folder on your Mac

How does the internet work?

For readers at home: this chapter is covered in the How the Internet Works video. This chapter is inspired by the talk "How the Internet works" by Jessica McKellar.

We bet that you use the Internet every day. But do you know what happens when you type an address like https://djangogirls.org into your browser's address bar and press enter?

This section will take you through some of the basics of how the Internet works so that you feel more confident when talking about Internet technology.

What is a website?

A website is a collection of files saved on a computer elsewhere in the world. These files could include pictures, videos and other stuff which you can download but all of this information is presented in web pages. A web page is a file consisting of Hyper Text Markup Language.

HTML is not a programming language but is instead used to create webpages which a web browser such as Chrome or Internet Explorer can read, load and display.

Where are these web files stored?

Whenever you go to a webserver such as https://djangogirls.org the pages which you view from that website are stored on their webserver. A webserver is simply a more powerful computer called a server which is connected to the Internet. The djangogirls server is located in California!

What is the command line?

The window, which is usually called the command line or command-line interface, is a text-based application for viewing, handling, and manipulating files on your computer. It's much like Windows Explorer or Finder on the Mac, but without the graphical interface. Other names for the command line are: cmd, CLI, prompt, console or terminal.
